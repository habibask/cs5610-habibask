Experiments

Worked with a third party open source script that provides the method contenthover() with which we can overlay one element over other on mouse over event and also decide on the animation effect and opacity.

Code Snippet

                   $(function () {
    $('#d1').contenthover({
        overlay_background: '#000',
        overlay_opacity: 0.8,
        effect: 'slide',
        slide_speed: 500,
        slide_direction: 'left'
    });
    $('#d2').contenthover({
        effect: 'slide',
        slide_speed: 300,
        overlay_background: '#000',
        overlay_opacity: 0.8
    });
});

The code snippet shows how the calls to the contenthover are made and the div element in the class contenthover is used to overlay on the element with which the event is triggered.
