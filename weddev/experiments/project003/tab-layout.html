Experiments

A tabbed layout where there is a tab for
1. User Introduction - About the user and the interests
2. History - the rides the user entered or he added to share
3. People - All the people the user shared a ride so far with
4. Settings - a page where the user can see the information entered during registration

Code Snippet

                  app.controller("profilecontroller", function ($scope, $sce, $http, $location, LoginService) {
    $(".tabs-menu a").click(function (event) {
        event.preventDefault();
        $(this).parent().addClass("current");
        $(this).parent().siblings().removeClass("current");
        var tab = $(this).attr("href");
        if (tab == '#tab-2') {
            var rides = LoginService.getRides(function (res) {
                console.log("Inside getrides in controller");
                console.log(res);
                $scope.rides = res;
            });
        }
        $(".tab-content").not(tab).css("display", "none");
        $(tab).fadeIn();
    });
}

In this code the we can see a controller assigned to the profile page. When we click on each tab menu respective page is open which is available in the profile.html
