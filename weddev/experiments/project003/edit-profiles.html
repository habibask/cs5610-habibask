Editing the user information entered during registration

Click on the button to navigate to the register page in the application hosted on openshift

Edit profiles

Experiments

A pencil icon is shown beside each group of information
When a pencil button is clicked the information becomes editable and the user can enter infromation
Also the pencil button becomes a ok button to click once the update is completed.
When ok is clicked the save button is display at then end of the page to push information to the database.
