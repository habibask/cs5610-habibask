An application with multiple navigations at the top using the angular routing concept

In this experiment, I experimented with the concept of angular-routing by creating 3 seperate files about.html, home.html and contacts.html.
The pages are then linked with the routing and shown in the container of the main page using ng-view attribute, rather than just creating the hyperlinks
