Image gallery

Experiments

An application which takes the urls of the images as input and displays them in other page as gallery.

for example try with url of an image "http://animalia-life.com/data_images/cat/cat2.jpg" In this experiment, I worked with the concept of angular-routing and services along with the controller functions by creating 2 seperate files home.html and gallery.html.
The pages are then linked with the routing and shown in the container of the main page using ng-view attribute.
There is a function defined in the pages in the gallery tab. Whenever a user adds image url in the home tab the imae is added to gallery by making use of a img tag.

Code Snippet


The code snippet shows how a application is configured for each relative url. There are different cases specified for different urls and the default case is mentioned in otherwise clause. The Service is defined using app.factory() method and then used to make calls from the galleryController. The updated array is being set to the urls array in the gallery using a method in controller.
