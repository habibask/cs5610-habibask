Code Snippet

In AngularJS scope watch

                   $scope.$watch(
        function () {
            return LoginService.getCurrentUser();
        },
       function (currentUser) {
           $scope.currentUser = currentUser;
           if (currentUser != null) {
               if (currentUser.userType == 'driver') {
                   $scope.isDriver = true;
               } else {
                   $scope.isDriver = null;
               }
           }
           console.log("controller current user " + currentUser);
       }, true);

Here when the user is verified for the login status the user information from the db is retrieved.
The user information has a field "userType" which contains if the user is of type driver or a sharer and the field isDriver is set accordingly which is used in ng-show of the main menu in AngularApp
